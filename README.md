# mapify
Web Application using Geolocation API and Leaflet map

## 💁 Features / Functionalities
  
  - Routing system
  - Current Location
  - Searching place with name
  - Searching places using latitude and longitude
  - places link
  - Add custom places
  <br>
  

## 🛠 Languages and Tools :
  - Geolocation API
  - Leaflet Map
  - javascript
  - HTML
  - CSS
  <br>
